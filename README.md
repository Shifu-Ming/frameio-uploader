# 🎬 Frame.io 上傳小精靈

下班前設定一次，下班後 Mac 自動傳影片到 Frame.io，不用守著電腦等上傳。

---

## 📦 下載

👉 **[點這裡到 Release 頁面](https://github.com/Shifu-Ming/frameio-uploader/releases/latest)**

到 Release 的 **Assets** 區下載這兩個檔：

1. **`Frame.io 上傳小精靈.app.zip`** ─ 主程式
   - 雙擊解壓 → 把 `.app` 拖到「**應用程式**」資料夾
2. **`reset.command`** ─ 卡住想重來時用（[詳見下方](#-卡住了想重來resetcommand)）
   - 隨意放（譬如桌面），需要時雙擊

> ⚠️ **不要下載 `Source code (zip)` 或 `Source code (tar.gz)`** ─ 那是 GitHub 自動加的程式碼快照，不是你要的。

> 前提：電腦要有 **Google Chrome**（沒有先裝 https://www.google.com/chrome ）

---

## 第一次設定

> ⚠️ **第一次打開要在 Finder 對 .app 按右鍵 →「打開」**（直接雙擊會被 macOS 擋）

雙擊 .app → 瀏覽器自動跳出設定介面。

1. 點紅色閃爍的「🔐 登入 Frame.io」
2. **跳出「自動化權限」彈窗 → 點「允許」**
3. Chrome 開啟 → 用你的 Adobe 帳號登入 Frame.io
4. 看到專案列表後，**自己關閉 Chrome 視窗**（重要：不關閉介面狀態不會更新）
5. 介面右上角會變綠「✓ 已登入」

### ⚠️ 第一次執行前一定要手動跑一次

第一次「▶️ 立刻執行一次」時，macOS 可能會跳這些彈窗，**全部點「好」/「允許」**：

- 「想存取**桌面**」/「下載項目」/「文件」上的檔案
- 「想存取**卸除式磁碟區**」（外接 SSD 用得到）

允許過一次之後就不會再問。**如果你還沒允許就直接設排程，當晚 Mac 會跳對話框等你點 → 排程會被卡住**。

所以順序一定是：
1. 設定完網址、資料夾、時間
2. **手動點「▶️ 立刻執行一次」測試**（順便允許所有權限）
3. 確認上傳成功
4. 之後才設排程

> 沒看到自動化權限彈窗：
> 系統設定 → 隱私權與安全性 → 自動化 → 找「Frame.io 上傳小精靈」勾選 Google Chrome

## 每天怎麼用

下班前（30 秒）：

1. 雙擊 .app
2. 貼**上傳目的地網址**（在 Frame.io 進到要上傳的資料夾，複製網址列）
3. 填**待上傳資料夾**（影片輸出位置）
4. 設**排程時間**（如 22:00）
5. 點「📅 儲存並排程」
6. 走人 ─ **介面可以直接讓它開著**

> 介面開著也好、關掉也好，排程都會照常執行（排程跟介面是分開的）。
> 想關的話點介面內的「🚪 關閉介面」按鈕（不要直接關 Chrome 分頁）。

> Mac 不可關機 / 進入睡眠（顯示器睡眠 OK），SSD 要連著。
> 系統設定 → 顯示器 → 「插電時不要進入睡眠」打勾。

## 🆘 卡住了想重來（reset.command）

從 [Release 頁面](https://github.com/Shifu-Ming/frameio-uploader/releases/latest) 下載 `reset.command`，**隨意放在桌面或哪裡都行**，需要時雙擊執行。

雙擊後會做這四件事：

1. 殺掉跑著的介面（如果還開著）
2. 取消所有排程
3. 清掉登入狀態 + 所有設定
4. 清掉 log

執行完之後雙擊 .app 會像「第一次使用」 ─ 重新登入、重新設定。**不影響 .app 本身**，只清資料。

適用情境：

- 登入流程卡住沒辦法繼續
- 想換成另一個 Adobe 帳號登入
- 介面狀態怪怪的，整個重來比較快
- 排程怎麼設都不對

## 常見問題

| 問題 | 解法 |
|---|---|
| 想立刻測試 | 點「▶️ 立刻執行一次」 |
| 取消今天的排程 | 點「❌ 取消排程」 |
| 上傳失敗想看細節 | 點「📂 開啟記錄資料夾」看最新 log |
| 突然顯示「未登入」 | Cookie 過期了，點「🔐 重新登入 Frame.io」 |
| 同名檔案 | 工具會自動把新檔合併成舊檔的新版本 |
| Frame.io 上沒看到自動切到 List | 不影響上傳，但同名版本迭代需要 List 模式。可手動到 Frame.io 切（Appearance → List）一次後它會記住 |

## 找誰

問題請找 **Ming**，順便附上記錄資料夾內最新那個 `upload_*.log`。
